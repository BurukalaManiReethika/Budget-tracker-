# 💰 Budget Tracker

A clean, dark-themed budget and expense tracker web app — built with pure HTML, CSS, and JavaScript. No frameworks, no installs, no signup needed.

🔗 **Live Demo:** https://burukalamanireethika.github.io/Budget-tracker-/

---

## ✨ Features

- **Set a monthly budget** with a live progress bar
- **Add expenses** with name, amount, and category
- **Real-time donut chart** showing spending by category
- **Color-coded alerts** — green → orange → red as you approach your limit
- **Delete expenses** anytime with one click
- **Works offline** — no backend, no database, just open the file!

## 🚀 How to Use

1. **Clone or download** this repo
2. Open `index.html` in any browser
3. Set your monthly budget
4. Start adding expenses
5. Watch your spending breakdown update live!

No server needed. Works on Chrome, Firefox, Edge, and Safari.
