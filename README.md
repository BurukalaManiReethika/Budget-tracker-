## вңҰ What Is This?

**Budget.** is a lean, dark-themed expense tracker that lives entirely inside a single HTML file. It was born from a simple frustration: most budgeting tools are either bloated apps demanding your email or ugly spreadsheets that don't spark any joy.

This is neither. It's fast, visual, and brutally honest about where your money goes.

## вңҰ Getting Started

### Option 1 вҖ” Use Instantly (Recommended)

Click the link. No setup. No install.

```
https://burukalamanireethika.github.io/Budget-tracker-/
```

### Option 2 вҖ” Run Locally

```bash
# Clone the repository
git clone https://github.com/BurukalaManiReethika/Budget-tracker-.git

# Navigate into it
cd Budget-tracker-

# Open in your browser (no server needed)
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

That's it. No `npm install`. No build step. No environment setup.
